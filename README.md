# üìà Python Stock Visualizer

An interactive web application for analyzing and visualizing historical stock market data. Built with **Python** and **Streamlit**, this tool allows investors to assess price trends, calculate technical indicators, and evaluate correlations between companies.

## üöÄ Key Features

* **Interactive Dashboard:** Compare stock prices of multiple companies (e.g., AAPL, GOOG, TSLA) over custom date ranges.
* **Market Analysis:** View **Top Gainers & Losers** with real-time percentage changes.
* **Correlation Heatmap:** Visualize how closely different stocks move together using an interactive matrix (powered by Altair).
* **Technical Deep-Dive:** Analyze individual stocks with **Candlestick Charts** and fundamental data (Sector, P/E Ratio).
